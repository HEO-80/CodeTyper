## вңЁ What is CodeTyper?

</div>

Most typing trainers give you *words*. CodeTyper gives you **real code**.  
As you type, every character you complete lights up with its **semantic color** вҖ” just like your IDE вҖ” so your brain starts associating muscle memory with code structure automatically.

## рҹҡҖ Features

</div>

- **вҢЁпёҸ Code-first typing trainer** вҖ” practice real, useful snippets, not toy examples
- **рҹҺЁ Live semantic highlighting** вҖ” colors update in real time as you type: keywords, strings, classes, functions, operators...
- **вқҢ Error detection** вҖ” wrong character? It flashes red and waits. No skipping allowed.
- **рҹ“Ҡ Session metrics** вҖ” CPM, accuracy percentage, total errors, and time
- **рҹҸҶ Grading system** вҖ” get rated S / A / B / C / D based on your accuracy
- **рҹ—ӮпёҸ Multi-language support** вҖ” JavaScript, Python, TypeScript, SQL (more coming)
- **рҹ“ҡ Snippet library** вҖ” curated snippets organized by language and difficulty
- **рҹ”„ Repeat or continue** вҖ” replay the same snippet or pick a new one after each session

## рҹ“Ұ Getting Started

</div>

```bash

cd codetyper

# 2. Install dependencies

# 3. Start the development server

```
